# 7FA4 Uploader
## 功能
上传任何文件（包括稀奇古怪的文件，诸如PDF、PPTX）  
还可以下载你上传的文件，只需输入上传时返回的提交ID  
同时，如果不想让别人看到你的文件，可以使用加密密码（注意不要忘了）
## 使用方法
在终端使用7FA4-Uploader.exe -h查看
## 限制
只能上传一定大小的文件  
至于大小嘛……你只能看有没有文件过大了  
加密的密码长度需为16、24或32字节（中文算2个）
## 安全
不会保存你输入的Cookie和密码  
但是在JX使用要小心有人“偷梁换柱”，把程序改了  
另外，使用powershell会保存历史记录（……）
